# Plugin Cadeia

> Esse repositório contém uma extensão desenvolvida para o jogo Unturned. Apartir dela é implementado a mecânica de prender e liberar jogadores da prisão, pré configurada, como também de chamar a policia apartir de comandos.

## ⚠️ Aviso

O projeto foi desenvolvido unicamente com fins recreativos, não há previsão para atualizações, futuros reparos nem qualquer garantia de qualidade ou segurança! Utilize a aplicação por sua conta e risco.
